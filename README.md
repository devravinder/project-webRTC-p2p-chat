# P2P Server-less WebRTC Chat

A Zoom/Teams-style P2P Server-less WebRTC app for chat, audio/video calls, screen share, screen recording, file sharing

Hoisted on Github pages - visit [Meet](https://chat.paravartech.com/) and use it for personal chat & also available as PWA.

## How to use

1. Open/vist [Meet](https://chat.paravartech.com) app and enter your name (first visit only).
2. To start a meeting, tap `Start Meeting`, pick max participants, and share the generated code/link with others.
3. To join, tap `Join Meeting` and enter the code you received (or just open the shared link).
4. Once connected, use the control bar to chat, mute/camera toggle, share your screen, or record.
5. Tap `Leave` to exit — closing the tab works too.
